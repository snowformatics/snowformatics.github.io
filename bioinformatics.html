si-Fi

si-Fi is a Software for RNAi (RNA interference) design and off-target prediction. The software is designed for checking long-double stranded RNAi constructs for target specificity and silencing efficiency. It can be used also to select sequence regions suitable for designing RNAi constructs. The program provides clear tabular and graphical outputs.

Primer Factory

Primer Factory is a primer design and primer search tool. It provides a convenient interface for the widely used command line software Primer 3 for designing PCR primers. PrimerFactory also can search by target sequence query in a primer stock database for matching primers that may eventually be reused instead of designing and ordering new oligonucleotides.
